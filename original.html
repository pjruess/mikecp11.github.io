Select route name: to

  • Select a route name from the drop down.
  • Click 'Add stops', then click on map to add stops to route.
  • Click the 'Add Barriers', then click on map to add barriers.
  • Optionally, repeat the above steps to add more routes.
  • Click the 'Solve Routes' to solve.

    • Any server error messages will be displayed in an alert box.
